## PTDC - Dashboard 


## Quick start for Development 

- Install dependencies: `npm install` 

- Start Application: `npm start`

- Views are on: `localhost:3000`

- Start the server: `npm start`

- Run are on: `localhost:5000`


## Quick start for Production 

- Install dependencies: `npm install` 

- Start Application: `npm start`

- Views are on: `localhost:3000`

## For build

- npm run build
